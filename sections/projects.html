Catering Website Development

Summary: Designed and developed a catering-themed website, starting from UI/UX design in Figma, then building the front-end using HTML, CSS, and JavaScript, and implementing a simple backend in Java to support interactive features.

Type: Class assignment (Individual Project)

Role & Impact: Worked on the entire process, from initial design to final implementation. The result was a functional website that allowed users to browse catering menus and place orders online.

Learning Outcome: Learned the end-to-end workflow of web application development, importance of user experience, and basics of integrating front-end and backend.

Smart Traffic Management System (STMS)

Summary: Developed an AI-powered traffic management solution integrating YOLOv5, CNN, Reinforcement Learning, IoT devices, and smart sensors to monitor traffic in real time and optimize light control.

Type: Class assignment (Group Project)

Role & Impact: Defined and implemented the Region of Interest (ROI) for traffic analysis, integrated YOLOv5 for object detection, and evaluated performance using simulation data.

Learning Outcome: Learned how to apply AI and computer vision techniques, evaluate performance, and collaborate effectively in a multidisciplinary team.

Driver Monitoring System (Prototype)

Summary: Developed a prototype of a real-time driver monitoring system designed to detect whether a driver is intoxicated, drowsy, or sober using CNN + LSTM models with TensorFlow/Keras and OpenCV.

Type: Class assignment (Group Project)

Role & Impact: Contributed to dataset collection, CNN + LSTM design, preprocessing, augmentation, and AI model training, as well as web interface and documentation.

Learning Outcome: Learned to apply deep learning to real-time data, importance of preprocessing and teamwork in AI projects.
